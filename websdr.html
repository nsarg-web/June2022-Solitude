WebSDR Software-Defined Radio is a webbased receiver, allowing many listeners to listen and tune it simultaneously. SDR technology makes it possible for all listeners to tune independently, and thus listen to different signals.

There are active receivers around the world, Some of them are provided by radio enthusists, there are several locatated at famous listening stations. Jodrell Bank in Cheshire, Goonhilly Earth Station in Cornwall. allowing a user to tune the receiver in real time, as if they were sitting infront of the radio at the selected location. Part of the Activity badge requirements the participant is required to log a number of Amateur Radio Transmissions. Using one of the many Web SDR receivers they can gain experiance with tuning the various receivers around the world to complete the logging requirements.

A list of active receivers can be found on the websdr organisation website at Websdr there are currently over 100 stations in the List. User simply select the receiver Location from the active stations. The list of stations can be filtered to show stations in a geographical area. Using WebSDR the user is able to view the signals received at one location Then view a different receiver from another station located in a different part of the Globe
